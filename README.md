StatusFeed
====

A status feed app using appbase.io , twitter API and ReactJS.

---

###Getting Started###



####Familiar with Git?#####
Checkout this repo, install dependencies, then start the gulp process with the following:

```
	> git clone https://github.com/Cst2989/StatusFeed.git
	> cd StatusFeed
	> npm install
	> gulp
```
